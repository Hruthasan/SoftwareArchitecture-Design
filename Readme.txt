Step1:Project Setup
Make a folder for your project. This will be your Django application's primary directory.


Step2:Django Installation
In your command prompt, type pip Â install Django. Before proceeding, ensure that Python and pip are installed on your system.


Step3:Running the server
Navigate to the directory containing your manage.py file. In your command prompt, type "python manage.py runserver" to start the Django server.

Step4:Accessing the application
When the server is properly started, an HTTP address "http://127.0.0.1:8000/" will be presented on your command prompt.

Step5:Viewing the application
Copy and paste the HTTP URL into your computer browser. If your Django application is successfully configured, the "Hello world!" message should appear on the webpage.
